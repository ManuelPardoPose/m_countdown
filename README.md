# M_COUNTDOWN

## Author: Manuel Pardo Pose

## How to build:
#### First:

    cd m_countdown


#### Then:

    cargo run --

    or

    cargo run -- --min 2

    or

    cargo run -- --min 2 --sec 10

    or

    cargo run -- --min 2 --sec 10 --bounce

    or

    cargo run -- --min 2 --sec 10 --ascii-mode

    or

    cargo run -- --bounce --ascii-mode

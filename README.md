# m_countdown

## Author
    Manuel Pardo Pose

## What is this?
    This is a CLI Application that provides a customizable countdown
    which is supposed to be lightweight and presentable.

## How to build:
#### First:

    cd m_countdown


#### Then:

    cargo run --

    or

    cargo run -- -h

    or

    cargo run -- -m 4 -s 20 -b -a --col1 200,100,20 --col2 255,0,0 --col3 50,50,200
